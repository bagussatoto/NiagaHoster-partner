== Description ==

Niagahoster Orderflow Plugin is a set of tools that will help you to sell domain and hosting from your website.

== Installation ==

Upload the Niagahoster Orderflow plugin to your blog, activate it, and then enter your Niagahoster Partner API key.

1, 2, 3: You're done!
